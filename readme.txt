Offers202 v1.0.0

Setting up Offers202 on your website is simple.  Please follow the steps below to install Offers202 on your own website.

1.	Download the files here: http://downloads.tracking202.com/offers202
2.	Extract the Offers202 Zip File.
3.	Take all of the files, and upload them into a folder on your website.
4.	Get Your Tracking202 API key.  You can get it by going to: http://developers.tracking202.com?redirect=developers
5.	Edit the offers202-config.php file and put your Tracking202 API key in there.  You will see something that says pasteYourApiKeyHere.  Replace that text with your Tracking202 API key.

You should now be able to run Offers202 on your own website.  You can modify your website however you wish.  Although you do not have to include a link to Offers202 on your site, we’d appreciate if you had one on. 

If you have any questions, please visit our developer’s forum.  You can find that forum here: http://prosper.tracking202.com/forum/
